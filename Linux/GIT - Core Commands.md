GIT - Core Commands

Clone:
```bash
git clone URL
```
Add files:
```bash
git add file
git add .      # everything
```
Commit:
```bash
git commit -m "message"
```
Push:
```bash
git push
```
Status:
```bash
git status
```
Ignore files:
```bash
touch .gitignore
```
Branches:
```bash
git branch new-feature
git checkout new-feature
```

**Pull Request**
Create a merge request via GitHub / GitLab UI.

**Environment Variables**
```bash
export passw='admin'
set | grep passw
```
