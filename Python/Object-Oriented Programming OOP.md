В класі об"єднують фукції які правильно називати методами! 
(self) - вказує на параметр класу 
print(dir(my_first_class)) dir - показує всі доступні методи для класа 
- `class` — створення класу
- `__init__` — конструктор
- `self` — посилання на об’єкт
- `dir(obj)` — показати методи об’єкта
- `super()` — звернення до батьківського класу
- `@dataclass` — автоматичне створення класу без `__init__`
- Приватна змінна: `self.__age = 200`
- Наслідування:
```python
class B(Person):
    def __init__(self, name):
        super().__init__(name)

```

inheritance - тут приклад унаслідування іншого класу (Person)

super() — це вбудована функція, яка дозволяє з дочірнього класу (child) звернутися до методів або атрибутів батьківського класу (parent), не дублюючи код.

| Що ти маєш на увазі          | Як це називається в Python                  | Пояснення                                                 |
| ---------------------------- | ------------------------------------------- | --------------------------------------------------------- |
| Порядок пошуку методів ----> | `MRO` — Method Resolution Order  ---->      | Визначає, з якого класу буде взято метод при наслідуванні |
| "Маркер" для зміни поведінки | Декоратор (`@staticmethod`, `@classmethod`) | Змінює спосіб виклику методу                              |
| Спеціальний порядок у класі  | Магічні методи (`__init__`, `__str__`...)   | Автоматично викликаються у певних умовах                  |
