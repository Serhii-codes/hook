В классах Class їх називають методами
- `def foo():` — оголошення функції
- `foo(3, 5)` — аргументи
- `(first_number=10, second_number=2)` — іменовані аргументи
- `exp(number, exponent=2)` — аргумент зі значенням за замовчуванням
- `return` — повертає значення
- `our_result = exp(abc(3, 5))` — виклик функції в іншій
