import re - імпорт бібліотеки 
output = re.findall('\w\w\w\w',list_a) - приклад патерну пошуку 4 символів в змінній list_a
w - шукає число ци цифру, нижнє підкреслення
W - навпаки шукає не число ци цифру, нижнє підкреслення
output = re.findall(r'\b[C-Z]\w{3}\b',list_a) - приклад пошуку по діапазону [C-Z]
.	будь-який символ, крім нового рядка	a.c → знаходить abc, a7c

- `re.findall()` — пошук збігів
- `\d` — цифра
- `\D` — не цифра
- `\w` — літера, цифра, _
- `\W` — все, крім `\w`
- `\s` — пробіл
- `\S` — не пробіл
- Email: `[\w\.-]+@[\w\.-]+\.\w+`
- Телефон (UA): `\+380\d{9}`
- IP: `\b(?:\d{1,3}\.){3}\d{1,3}\b`
- URL: `https?://[\w./%-]+`
