 Lists
- `[]` — створення списку
- `list[0] = "..."` — зміна елемента за індексом
- `names[0:3]` — зріз елементів (0,1,2)
- `numbers.append(6)` — додавання в кінець
- `numbers.insert(0, -1)` — вставка на позицію
- `.remove(1)` — видалення елемента
- `.clear()` — очищення списку
- `.pop()` — видалення останнього елемента
- `a[:]` — повний список
- `a[1:4]` — зріз по індексах
- `a[::-1]` — розворот списку
- `not in` — перевірка відсутності елемента

Dictionaries
- `.setdefault` — додає ключ зі значенням, якщо його ще немає
- `.update({"k": "v"})` — додавання пари ключ-значення
- `del dict["k"]` — видалення елемента
- `.popitem()` — видаляє останню пару
- `.values()` — всі значення
- `.keys()` — всі ключі
- `.items()` — ключі та значення разом
- `.get(key)` — безпечне отримання значення
