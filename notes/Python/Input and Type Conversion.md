- `input()` — введення з консолі
- `int()` — перетворення у ціле число
- `float()` — перетворення у число з плаваючою крапкою
- `a = int(input())` — збереження вводу як цілого числа
- `a = float(input())` — збереження вводу як дробовог числа
- `a = float(input("Введите значение a:"))` — підказка при вводі
- `a, b, c = map(int, input().split())` — введення кількох значень в один рядок
