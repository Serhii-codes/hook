- `python3 -m pytest -v` — варіант запуску (більш повний), як скрипт
- `pytest -v` — більш простіший варіант запсуку
- `pytest test_scope_1_with_class.py -v` — запуск конкретного файлу
- `pytest test_scope_1_with_class.py::TestNew -v` — запуск конкретного класу с тестами (::TestNew - назва класу) використовуємо `::` для кроку в середену
- `pytest test_scope_1_with_class.py::TestNew::test_1 -v` — запуск конкретного тесту (::test_1 - назва тесту в класі) використовуємо `::` для кроку в середену
