### **Що таке adjective**

Adjective = слово, яке описує іменник (noun)  
→ _big house, cold water, expensive phone_

**НЕ змінюється за родом / числом**
- one big car
- two big cars  
    → _big_ не міняється

### **Позиція прикметника**

|Type|Example|
|---|---|
|Перед іменником (атрибут)|a beautiful day|
|Після be / seem / look / feel (предикатив)|the day is beautiful|
### **Порядок кількох adjectives (ORDER)**
**Opinion → Size → Age → Shape → Colour → Origin → Material → Purpose + NOUN**

Example:
 a nice small old round black Italian leather travel bag

**Comparative & Superlative**
| Type                 | Rule                             |  Example                    |

| Comparative | порівнюємо A vs B | bigger / more interesting  |
| Superlative    | “най-” в групі            | biggest / most interesting |

**Основні правила утворення**:
- короткі прикметники → `+er` / `+est`  
    → big → bigger → biggest
- довгі прикметники → `more` / `most`  
    → interesting → more interesting → most interesting
**подвоєння приголосної**:  
big → bigger (бо коротке слово і закінчується на consonant + vowel + consonant)

**-ed vs -ing adjectives**
-ed	    описує почуття людини
-ing 	описує річ / подію (яка викликає емоцію)
Example:
- I am bored.
- This movie is boring.

### **Adjective ≠ Adverb**
- adjective описує **іменник**
- adverb описує **дію / adjective / sentence**
He is quick. → adjective  
He runs quickly. → adverb

### **Modifiers of adjectives**
- very big
- quite interesting
- a little cold

### **Adjectives after something / anything / nothing**
- something strange    
- anything new
- nothing important

**ВИНЯТКИ / IRREGULAR COMPARATIVES**

| Adjective   | Comparative       | Superlative         |
| ----------- | ----------------- | ------------------- |
| good        | better            | best                |
| bad         | worse             | worst               |
| far         | farther / further | farthest / furthest |
| little      | less              | least               |
| many / much | more              | most                |

**Adjectives that never take “more / most”** (тільки -er / -est)
- old → older → oldest
- young → younger → youngest
- small → smaller → smallest  
    (короткі 1–2 склади — правило)

**Adjectives that never take -er / -est** (тільки more / most)
- interesting
- important
- expensive  
    (довгі 3+ склади — правило)

**Adjectives з фіксованими значеннями (НЕ градуються)**
ці — не підсилюються very / more  
→ тільки absolutely / completely / totally
- perfect (не буває “more perfect”)
- unique
- dead
- impossible
- ideal
правильно: **absolutely unique**  
неправильно: very unique

### Comparison of Adjectives

 **1. Three degrees of comparison**
1️⃣ **Positive degree** — базова форма  
2️⃣ **Comparative degree** — порівняння (більш…)  
3️⃣ **Superlative degree** — найвищий ступінь (най…)

**Examples:**
- small → **smaller** → **the smallest**
- interesting → **more interesting** → **the most interesting**

 **2. Forming the Comparative**
**A) Short adjectives (1–2 syllables)**
Додаємо **-er**

|Adjective|Comparative|
|---|---|
|tall|taller|
|fast|faster|
|small|smaller|
**Правила написання:**
✔ **-e → +r**
- nice → nicer
- large → larger
✔ **Приголосна + y → y → i + er**
- happy → happier
- busy → busier

✔ **CVC (consonant–vowel–consonant): подвоєння останньої букви**
- big → bigger
- hot → hotter
- thin → thinner

---

 **3. Forming the Superlative**
 **Short adjectives**
Додаємо **the + -est**

|Adjective|Superlative|
|---|---|
|tall|the tallest|
|fast|the fastest|
**Правила як у comparative:**
- nice → the nicest
- happy → the happiest
- big → the biggest
- 
**4. Long adjectives (≥ 3 syllables)**
Використовуємо **more** / **most**

|Adjective|Comparative|Superlative|
|---|---|---|
|interesting|more interesting|the most interesting|
|expensive|more expensive|the most expensive|
|difficult|more difficult|the most difficult|
**5. Irregular adjectives (Неправильні)**

| Positive  | Comparative       | Superlative                 |
| --------- | ----------------- | --------------------------- |
| good      | better            | the best                    |
| bad       | worse             | the worst                   |
| far       | farther / further | the farthest / the furthest |
| little    | less              | the least                   |
| much/many | more              | the most                    |
 **6. Adjectives with two acceptable forms**

(May take **-er/-est** or **more/most**)
✔ common
- common → **commoner** / **more common**
- the commonest / the most common

✔ narrow, polite, simple, quiet
- simple → simpler / more simple
- polite → politer / more polite
> У сучасній англійській частіше використовують **more + adjective** для ввічливості та формальності.

**7. Two-syllable adjectives ending in –y, –ow, –er, –le**
Вони зазвичай утворюють ступені з **-er / -est**
- happy → happier → the happiest
- narrow → narrower → the narrowest
- clever → cleverer → the cleverest
- simple → simpler → the simplest

**8. Using “Than”**
У порівняннях **always use "than"**
- She is taller **than** me.
- This book is more interesting **than** that one.
- My car is less expensive **than** yours.

 **9. “Less” / “The least”**
Для всіх прикметників — short і long
- less interesting
- less big
- the least expensive
- the least important

**10. Double comparatives (Підсилення порівняння)**

**A) More and more / less and less**
- The weather is getting **colder and colder**.
- Life is becoming **more and more difficult**.

 **B) The + comparative, the + comparative**
- **The more** you practice, **the better** you get.
- **The less** you sleep, **the worse** you feel.

 **11. Comparison with nouns**
Іноді порівняння не про якість, а про кількість:
- I have **more** money than you.
- She has **the most** friends.
- They have **less** time this week.
- He has **the least** experience.

**12. Special structures**
 **A) as + adjective + as**
(Так само … як)
- She is **as tall as** her brother.
- The test wasn’t **as difficult as** I expected.
**B) not as/so + adjective + as**
(Не такий … як)
- This film isn’t **as good as** the first one.
- The room isn’t **so big as** yours. (BrE)

 **13. Common mistakes**
❌ _more better_ → ✔ **better**  
❌ _the most fastest_ → ✔ **the fastest**  
❌ _more nicer_ → ✔ **nicer**  
❌ _more big_ (incorrect for short adjectives) → ✔ **bigger**


**14. Quick summary (для швидкого повторення)**

| Type of adjective | Comparative  | Superlative  |
| ----------------- | ------------ | ------------ |
| Short (1–2 syll)  | -er          | -est         |
| Long (≥3 syll)    | more         | most         |
| Irregular         | special form | special form |
|                   |              |              |
