Це набір правил, протоколів та інструментів, який дозволяє різним програмам, сервісам або системам взаємодіяти, обмінюватися даними та командами між собою. API діє як посередник, дозволяючи програмам "спілкуватися", не знаючи деталей внутрішньої роботи один одного.
